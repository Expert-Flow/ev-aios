# context/

Ez a mappa rólad és a vállalkozásodról szól. A `/bevezetes` skill tölti fel a következő fájlokkal:

- `vallalkozas.md` — vállalkozás identitás, ICP, ajánlat (Q1 + Q3 alapján)
- `szemelyes.md` — személyes kontextus, értékek, motivációk (Q2 + Q3 alapján)
- `prioritasok.md` — Q3 alapján a következő 90 nap

Bármikor szerkesztheted ezeket kézzel — a `/bevezetes` újrafuttatása megkérdezi, hogy felülírja vagy egybeolvassza.
