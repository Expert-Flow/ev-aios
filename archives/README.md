# archives/

Régi dokumentumok és lezárt projektek temetője.

**Szabály**: SOSE töröld a kontextust. Mindig ide mozgasd. A `/attekintes` automatikusan kihagyja az archívumot a frissesség-ellenőrzésnél, de elérhető marad ha kell.

Tipikus tartalom:
- Lezárt ügyfél-projektek (a `projektek/` mappából)
- Régi verziójú `ev-intake.md` (ha újrastrukturálod)
- Levonta hasznát szövegminták, sablonok
- Leváltott eszközök dokumentációja

Nem kell formális struktúra — egy mappa év szerint elég. Pl. `archives/2026/`.
